1. Job Description

1.1 Buscamos un Desarrollador Full Stack Mid/Senior especializado en Python y FastAPI.

1.2 El proyecto consiste en construir y mantener una plataforma moderna basada en arquitectura distribuida, con procesamiento asíncrono, orquestación de flujos y capacidades de integración con modelos de lenguaje (LLMs).

1.3 El desafío será desarrollar APIs escalables, gestionar tareas en segundo plano, optimizar el rendimiento del sistema y colaborar con equipos técnicos para integrar servicios avanzados de IA y automatización.

2. Responsabilidades

  1. 2.1 Desarrollar y mantener APIs asíncronas con FastAPI y Python 3.
  2. 2.2 Diseñar y optimizar modelos de datos usando Beanie (ODM) sobre MongoDB.
  3. 2.3 Implementar tareas distribuidas y procesamiento en background con Celery y RabbitMQ/Redis.
  4. 2.4 Desarrollar e integrar flujos event-driven y sistemas basados en mensajería.
  5. 2.5 Escribir pruebas unitarias y de integración usando pytest, unittest.mock y contextlib.contextmanager.
  6. 2.6 Gestionar entornos de desarrollo y despliegue con Docker Compose.
  7. 2.7 Implementar integraciones con LLMs (OpenAI, Anthropic, etc.) y flujos de orquestación de prompts.
  8. 2.8 Monitorear, loggear y optimizar el rendimiento del backend en entornos cloud (AWS / GCP).
  9. 2.9 Colaborar con el equipo frontend para construir interfaces con React, Redux Toolkit, TypeScript y Tailwind.
  10. 2.10 Participar en la definición de arquitectura, revisión de código y mejora continua del stack técnico.

3. Requisitos obligatorios

  1. 3.1 3–5 años de experiencia en desarrollo backend con Python.
  2. 3.2 Experiencia sólida en FastAPI, Pydantic, asyncio y programación asíncrona.
  3. 3.3 Conocimiento en Celery, RabbitMQ o Redis para gestión de tareas distribuidas.
  4. 3.4 Experiencia con MongoDB y modelado de datos con Beanie u otros ODMs.
  5. 3.5 Sólida comprensión de arquitectura distribuida, event-driven design patterns, concurrencia y paralelismo.
  6. 3.6 Experiencia en testing automatizado (pytest, mocking, unit/integration tests).
  7. 3.7 Experiencia en Docker Compose y despliegues en AWS / GCP.
  8. 3.8 Familiaridad con workflow orchestration y message brokers.
  9. 3.9 Capacidad para integrar y consumir APIs de LLMs (OpenAI, Anthropic, etc.).
  10. 3.10 Conocimiento de control de versiones (Git) y metodologías ágiles (Scrum / Kanban).
  11. 3.11 Nivel de inglés B2/C1, con capacidad de comunicación técnica fluida.
  12. 3.12 Alta autonomía y capacidad de autogestión, capaz de proponer soluciones y avanzar sin supervisión constante.

4. Nice to Have

  2. 4.2 Experiencia con TypeScript, Zod, Axios, Jest, ESLint en pipelines o automaciones.
  3. 4.3 Experiencia en CI/CD pipelines y monitoreo de servicios.
  4. 4.4 Conocimientos básicos de conceptos LLM y herramientas de IA generativa.
  5. 4.5 Interés en la mejora continua y adopción de nuevas tecnologías cloud-native.

  1. 5.1 Experiencia mínima: 3+ años en desarrollo backend, 1+ en desarrollo frontend.
  2. 5.2 Capacidad de escribir código limpio y mantenible y revisarlo en PRs.
  3. 5.3 Alta capacidad de análisis y resolución de problemas complejos.
  4. 5.4 Comunicación efectiva con stakeholders técnicos y de negocio.
  5. 5.5 Experiencia en sistemas de misión crítica y alta concurrencia.
  6. 5.6 Inglés: B2+ (excluyente)
